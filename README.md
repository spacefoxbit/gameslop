# Flappy UFO (gameslop)

A simple Flappy Birdâ€“style game where you control a cute UFO dodging trees. Made with HTML5 Canvas and JavaScript.

## How to Play
- Press **Space** to start and to make the UFO flap upward.
- Avoid hitting the trees or the edges of the screen.
- Try to get the highest score!

## Run Locally
1. Download or clone this repository:
   ```sh
   git clone https://github.com/spacefoxbit/gameslop.git
   ```
2. Open `index.html` in your web browser.
